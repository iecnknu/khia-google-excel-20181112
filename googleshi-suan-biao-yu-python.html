Google試算表與Python

Google試算表與Python

Google試算表除了上述的優點，我們更能藉由它儲存在雲端的特性，藉由申請後的開發者權限遠端連結它，使之成為一個迷你的資料庫。

這邊我們將藉由Python程式語言來示範。

  1. 申請Google API

    1. 開啟Google Cloud Platform
    2. 開啟一個新的專案
    3. 在新的專案底下啟用Google Drive API和Google Sheet API
    4. 建立新的憑證
      1. 您目前使用哪個 API？：Google Drive API
      2. API 的呼叫來源為何？：網路伺服器
      3. 您需要存取什麼資料？：應用程式資料
      4. 您準備搭配 App Engine 或 Compute Engine 使用此 API 嗎？：否，我並未使用任何一項服務
      5. 點選【我需要哪些憑證】
      6. 服務帳戶名稱：自由選寫
      7. 角色：project > 編輯者
      8. 金鑰類型：JSON
      9. 下載產生的JSON檔，我們將依靠它連結Goolge試算表

  2. 共享試算表

    1. 找到剛剛下載的JSON檔，複製出client_email的資料，會是一個自動產生的email
    2. 將欲操作的Googleg試算表共享給此email

  3. Python

    1. 首先透過pip安裝 gspread 和 oauth2client
      oauth2clien是一個為python打造的認證庫。它讓我們的應用能透過多種方法通過認證，建立和伺服器之間的連線。
      pip install gspread oauth2client

    2. 開啟一個新的檔案，為方便日後操作，命名為【googleSheet.py】

    3. 匯入套件和模組

      import gspread
from oauth2client.service_account import ServiceAccountCredentials

    4. 啟動授權，匯入剛剛下載的JSON檔

      scope=['https://spreadsheets.google.com/feeds','https://www.googleapis.com/auth/drive']
creds = ServiceAccountCredentials.from_json_keyfile_name('JSON檔案路徑',scope)
client = gspread.authorize(creds)

    5. 打開目標試算表的第一個工作表(sheet 1)，並指派給一個sheet物件。

      sheet = client.open('試算表名稱').sheet1

    6. 參考gspread的官方文件，尋找需要的函式。 即可以透過Python控制該工作表。 gspread官方文件
      gspread API文件

[warning] 注意

Google Cloud Platform雖然沒有限制每日的瀏覽量，但卻嚴格控管每一百秒可以對該試算表寫入和讀取的次數。
每一個專案每100秒能有500個請求，每一個使用者每100秒能有100個請求。

如果想要在100秒內有更大量的請求次數，請至專案中選取要增加的API，點選左邊欄位的配額，選擇要修改的項目；預設已經是一般使用者的最高上限，如要提高上限，請申請增加配額(填寫申請單)。
