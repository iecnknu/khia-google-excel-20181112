SUM、AVERAGE、IFS、FREQUENCY

為了方便使用，這裡幫大家把四個待會兒會用到的函式列出來。

🔰SUM

傳回一連串數字或儲存格的總和。

使用範本

SUM(A2:A100)

SUM(1,2,3,4,5)

SUM(1,2,A2:A50)

語法

SUM(value1, [value2, ...])

  • 值_1要相加的第一個數字或範圍。
  • 值_2-[OPTIONAL]- Additional numbers or ranges to add to值_1.

🔰AVERAGE

傳回資料集的數字平均值 (忽略文字)。

使用範本

AVERAGE(A2:A100,B2:B100,4,26)

AVERAGE(1,2,3,4,5,C6:C20)

語法

AVERAGE(value1, [value2, ...])

  • 值_1計算眾數時要列入計算的第一個值或範圍。 平均價值
  • 值_2-[OPTIONAL]- Additional 計算平均值時要列入計算的其他值或範圍。

🔰IFS

計算多個條件並傳回對應第一個 True 條件的值。

使用範本

IFS(A1>90, "A", A1>80, "B", A1>70, "C")

IFS({A1:A5} > 3, "大於", {A1:A5} = 3, "等於")

語法

IFS(條件_1, 值_1, [條件_2, 值_2, …])

  • 條件_1- 第一個要評估的條件；可為布林值、數字、陣列或是上述任一項目的參照。
  • 值_1-條件_1為TRUE時傳回的值。
  • 條件_2, 值_2, …- 第一個條件的評估結果為 FALSE 時，用來評估的其他條件和值。

🔰FREQUENCY

計算單欄陣列的頻率分佈並且分成指定類別。

使用範本

FREQUENCY(A2:A40,B2:B5)

語法

FREQUENCY(資料, 類別)

  • data包含值的陣列或範圍個數。
  • 類別包含類別集合的陣列或範圍。
    • 類別 should be sorted for clarity, but FREQUENCY will sort the values specified internally if they are not and return correct results.
